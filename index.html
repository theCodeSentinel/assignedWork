CSS Position Properties

The position property in CSS determines how an element is positioned in a document. There are five values for the `position` property:

Static

  • Default value: Element is positioned according to the normal document flow
  • Top, right, bottom, and left properties have no effect

    position: static;

Relative

  • Element is positioned relative to its normal position
  • Top, right, bottom, and left properties specify the offset from its normal position
  • Other elements are not affected by the offset

position: relative;
top: 20px;
left: 30px;

Absolute

  • Element is removed from the normal document flow Positioned relative to its nearest positioned ancestor (instead of its parent)
  • Top, right, bottom, and left properties specify the offset from its ancestor
position: absolute;
top: 0;
right: 0;

Fixed

  • Element is removed from the normal document flow Positioned relative to the viewport (browser window)
  • Top, right, bottom, and left properties specify the offset from the viewport Element remains
fixed even when the page is scrolled position: fixed;
top: 0;
left: 0;

Sticky

  • Element is positioned based on the user's scroll position
  • Element is positioned relative to its nearest ancestor with a scrolling
box
  • Top, right, bottom, and left properties specify the offset from its ancestor
position: sticky;
top: 0;

CSS Float Properties

The `float` property in CSS specifies whether an element should float to the left or right of its parent element.

  • `left`: Element floats to the left of its parent
  • `right`: Element floats to the right of its parent
  • `none`: Default value, element does not float
float: left;
float: right;
float: none;
clear: left;
clear: right;
clear: both;

Common use cases for float include:

  • Wrapping text around images
  • Creating multi-column layouts
  • Positioning elements outside of the normal document flow

However, modern CSS layout techniques like Flexbox and Grid have largely replaced the need for float-based layouts.
